# vue-ufp-ui-charts

## 概述

中信银行图表组件

## 安装

```bash
npm install echarts vue-ufp-ui-charts
```

## 配置

::: demo 在`ufp-framework.js`文件中添加如下配置

```javascript
module.exports = {
  plugins: [
    [
      'ui',
      {
        components: [
          ['charts', {
            'all|require': 'echarts/echarts.all',
            'theme|require': 'echarts/theme/macarons',
            'china|require': 'echarts/map/js/china.js',
          }],
        ]
      }
    ]
  ]
};
```
