# ui-noticebar

## 概述

通告栏组件，通过属性配置，展示文字滚动效果

## 安装

```bash
npm i vue-ufp-ui-noticebar
```

## 配置
::: demo 在`ufp-framework.js`文件中添加如下配置

```javascript
module.exports = {
  plugins: [
    ['ui', {
      components: [
        'noticebar'
      ]
    }]
  ]
}
```
:::

## 使用
::: demo 具体配置请参考参数

```html
<template>
  <div>
    <h2>通告栏</h2>
    <cb-notice-bar>
      <div>中信银行是中国改革开放中最早成立的新兴商业银行之一</div>
    </cb-notice-bar>
    <h2>通告栏（不显示图标）</h2>
    <cb-notice-bar :icon-show="false">
      <div>中信银行是中国改革开放中最早成立的新兴商业银行之一</div>
    </cb-notice-bar>
    <h2>通告栏（关闭按钮）</h2>
    <cb-notice-bar
      :close-button-show="true"
      @closeButtonClick="closeButtonClickHandler">
      <div>中信银行是中国改革开放中最早成立的新兴商业银行之一</div>
    </cb-notice-bar>

    <h2>通告栏（自定义颜色）</h2>
    <cb-notice-bar
      background-color="#409eff"
      font-color="#ffffff">
      <div>中信银行是中国改革开放中最早成立的新兴商业银行之一</div>
    </cb-notice-bar>


    <h2>通告栏（滚动）</h2>
    <div>
      <cb-notice-bar
        :speed="800">
        <div>中信银行成立于1987年，是中国改革开放中最早成立的新兴商业银行之一，是中国最早参与国内外金融市场融资的商业银行</div>
      </cb-notice-bar>
    </div>
    <div class="mgt-20">
      <cb-notice-bar
        :close-button-show="true"
        :speed="1200">
        <div>中信银行成立于1987年，是中国改革开放中最早成立的新兴商业银行之一，是中国最早参与国内外金融市场融资的商业银行</div>
      </cb-notice-bar>
    </div>
  </div>
</template>
<script>
export default {
    data() {
      return {
        title: 'NoticeBar 通告栏'
      }
    },
    methods: {
      closeButtonClickHandler() {
        this.$CBPlugin.toast.show('关闭按钮点击了');
      }
    }
  }
</script>
:::
### 参数

#### 属性说明
| 属性名称      | 数据类型   |必输  | 默认值   | 说明  |
|:-------------:|:------------:|:--------:|:--------:|:-----:|
| background-color | String | N|  #fefcec |设置背景颜色<br/>颜色值可以传任意值|
| font-color | String | N|  #f76a24 |设置字体颜色<br/>颜色值可以传任意值|
| icon-show | Boolean | N|  true |设置喇叭图标是否显示<br/>默认显示，若不显示传false|
| close-button-show  | Boolean | N| false | 关闭按纽是否显示<br/>默认不显示，若显示传true |
| speed  | Number | N| 0 | 滚动速度，为0不滚动 |

#### 插槽说明
| 插槽名称      |  说明  |
|:-------------:|:--------:|
| 默认插槽 | 该插槽内容为通告栏内自定义内容 |

#### 事件说明
| 事件     | 参数 | 说明  |
|:---------:|:----:|:--------:|
| closeButtonClick |- |点击关闭按钮，操作事件处理 |
