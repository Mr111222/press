# ui-sticky

## 概述

吸顶容器组件，用于页面滑动时，标题粘滞效果实现

## 安装

```bash
npm i vue-ufp-ui-sticky
```

## 配置
::: demo 在`ufp-framework.js`文件中添加如下配置

```javascript
module.exports = {
  plugins: [
    ['ui', {
      components: [
        'sticky'
      ]
    }]
  ]
}
```
:::
